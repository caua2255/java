CRIAR UM NOVO REPOSITÓRIO

1.     Configurando o git:
- git config user.name "Caua2255"
- git config user.email "caua.louzeiro@sempreceub.com"


2.     Baixar o git na pasta:
- git init
- git config user.name
- git config user.email

3. Verificar status de mudança:
- git status

4. Adicionar tudo:
- git add .
- git status
- git commit -m "Meu primeiro salvamento oficial"

5. Monitorar histórico do código:
- git log
- git log --oneline
- para sair da tela "Q"

cntrol L limpa tela, cntrl C para o código "powershell"
já no git "só aperta 'Q' ".

6. Mandar para o Github:
- Criar o repositório no Github primeiro!
- git branch -M main "Entrar na principal"
- git remote add origin https://github.com/caua2255/java.git
- git branch -M main


7. Eviar os commits para o Github:
- git push -u origin main

7. DESLOGAR:
- git config --global --unset credential.helper



clonar o repositório:
- git clone https://github.com/caua2255/POO_Java.git

- git config user.name
- git config user.email


Adicionar mudanças:
- git add .
- git commit -m "mudança"

Conectar remotamente ao github:
- git remote add origin https://github.com/caua2255/POO_Java.git



Limpar os dados do Git dentro do VScode:
- git config --global --unset-all user.name
